# GRAIL

GRAIL Collectibles LLC marketplace built with Next.js App Router, Supabase, Stripe, and Vercel.

Production domain:

https://www.grailcollects.com

## Local Development

```bash
npm run dev
```

Open http://localhost:3000 in your browser for local development only.

## Site Configuration

Production site URLs, canonical URLs, Open Graph defaults, and support email are centralized in:

`app/lib/siteConfig.ts`

Change the production domain there if GRAIL moves domains in the future.
